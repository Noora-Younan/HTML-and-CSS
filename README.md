# NOORA YOUNAN // WEB DEV

**STRATEGY. DESIGN. EXECUTION.**
This repository hosts two distinct web projects demonstrating proficiency in HTML5, CSS3, and Bootstrap 4.

## PROJECT 01: CINEMA PREMIERE (BOOTSTRAP)
A high-impact movie listing interface utilizing the **Bootstrap 4** library.

* **THEME:** Modern Red & White (Netflix-style branding).
* **STRUCTURE:** * Utilizes `jumbotron-fluid` for full-width visual impact.
    * Responsive `navbar-expand-lg` with danger-theme styling.
* **DATA:** * Implementation of Card Decks for grid layout.
    * Integration of Pill Badges for scheduling data.
    * Accurate sourcing of external image assets.

## PROJECT 02: BRUTALIST PORTFOLIO
A hard-coded HTML/CSS portfolio emphasizing stark contrast and structural layout.

* **DESIGN LANGUAGE:**
    * **Typography:** Bold, uppercase fonts (Impact/Arial Black).
    * **Palette:** High contrast Black/White/Red.
    * **Borders:** Thick, visible borders defining the grid.
* **TECHNICAL SPECS:**
    * **Fixed Menu:** Sticky navigation with high-contrast hover states.
    * **CSS Filters:** Inversion and high-contrast filters applied to imagery.
    * **Grid System:** Two-column split layout.
    * **Video Integration:** Grayscale video background with solid block text overlays.
    * **Data Visualization:** Styled HTML tables with strict border styling.
